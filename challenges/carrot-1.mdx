## Overview

The Carrot 1 challenge is our express evaluation. One phase, one profit target — built for experienced traders who want to get funded without the two-phase process. In exchange for the speed, the profit target is slightly higher and the drawdown limits are tighter.

## Phase parameters

### Phase 1 — Evaluation

| Parameter | Value |
|---|---|
| Profit Target | 10% |
| Max Daily Drawdown | 4% |
| Max Total Drawdown | 8% |
| Minimum Trading Days | 3 |
| Time Limit | None |

## Funded account conditions

| Parameter | Value |
|---|---|
| Max Daily Drawdown | 4% |
| Max Total Drawdown | 8% |
| Profit Split | Up to 90% |
| Payout Frequency | Bi-weekly |
| Scaling Available | Yes |

## Carrot 1 vs Carrot 2 — which is faster?

Mathematically, Carrot 1 gets you funded in fewer trading days — but the tighter drawdown rules mean less room for error.
